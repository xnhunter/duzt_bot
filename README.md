Overview
--------
- **Name**: UkrDUZT Bot
- **Version**: 1.0.0
- **Release**: Alpha

Build
-----------
- Create student_teacher_list.txt and fill it in, template: [Name] [Last Name] [Link]
- Initiate bot_token variable with your own in duzt_bot.py;
- Run the script in a Python interpreter;
- Start the bot in Telegram.
